Python有五个标准的数据类型：

- Numbers（数字）有整型 浮点型 复数
- String（字符串）
- List（列表）可以看作数组
- Tuple（元组）可以看作只读的列表
- Dictionary（字典）可以看作Map

字符串

str = 'Hello World!'

print str[2:5]

列表

list = [ 'runoob', 786 , 2.23, 'john', 70.2 ]

元组

tuple = ( 'runoob', 786 , 2.23, 'john', 70.2 )

字典

tinydict = {'name': 'runoob','code':6734, 'dept': 'sales'}
